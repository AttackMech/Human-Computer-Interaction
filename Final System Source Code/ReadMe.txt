This project was developed using Visual Studio Community 2015
To run, open the solution and build.  Make sure to run the the x64 debug version.
The program is dependent on the Allegro game library.  The library files are included in the project directory, and should be linked correctly in the project properties.

There are no special logins or passwords required.  All inputs are simulated via the control structures to the right.  Click on the controls to simulate input.  To begin, use the console controls on the right of the screen to turn on the device.  The screens presented in the UI are controlled via the controller UI to the right.  Additionally, the console controls and game discs can be accessed in a similar manner.
